**Sales Data Analysis Project** 

This Power BI dashboard analyzes sales data, helping users track revenue growth, product performance and customer buying patterns over time through interactive visualizations.

    1. Sources / Tutorials / Data

This project was created following a tutorial playlist: [Codebasics Playlist](https://youtube.com/playlist?list=PLeo1K3hjS3uva8pk1FI3iK9kCOKQdz1I9&si=_hUiMtjbXMgH6Uvi) as a starting point, with additional dashboards, analyses & visualizations implemented by me as personal contributions. The sample dataset used can be found here: [Sales Insights Dataset](https://codebasics.io/resources/sales-insights-data-analysis-project) , it was provided by the tutorial author and simulates the activity of a computer hardware business in India interested in sales insights. 
No sensitive or real company data is included.
